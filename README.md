# Prática de Git e GitHub

## Descrição 
Este repositório foi criado para praticar os fundamentos de Git local antes de publicar projetos no GitHub.

## Objetivo
Aprender o fluxo básico de versionamento com Git, incluindo criação de commits, revisão de alterações, branches, merge e resolução de conflitos.

## Conteúdos praticados

- git init
- git status
- git add
- git commit
- git log
- git diff
- git restore
- .gitignore
- branches
- merge
- merge conflict

## Aprendizados

Este repositório funciona como um laboratório para entender Git de forma prática, antes de aplicar versionamento em projetos maiores.

## Próximos passos

- Criar um repositório remoto no GitHub
- Fazer o primeiro push
- Praticar clone
- Aplicar Git em um projeto Python real
